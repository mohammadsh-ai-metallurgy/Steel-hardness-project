# Steel-hardness-project
# پروژه تحلیل سختی فولاد AISI 4140

این پروژه با استفاده از زبان پایتون، داده‌های مربوط به عملیات حرارتی فولاد ۴۱۴۰ را تحلیل و نمودار سختی بر حسب دمای تمپر را رسم می‌کند.

## ویژگی‌ها
- خواندن داده‌ها از فایل اکسل.
- پیش‌پردازش داده‌های متالورژیکی.
- رسم نمودار استاندارد مهندسی با استفاده از Matplotlib.

## نحوه اجرا
1. کتابخانه‌های مورد نیاز را نصب کنید:
   `pip install pandas matplotlib openpyxl`
2. فایل اکسل خود را در مسیر صحیح قرار دهید.
3. کد `main.py` را اجرا کنید.

## پیش‌نیازها
- Python 3.x
- pandas
- matplotlib
- openpyxl
